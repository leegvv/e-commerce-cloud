# e-commerce-cloud
电商微服务

## 主要模块

### config
配置文件: checkstyle配置

### e-commerce-admin
微服务应用监控

### e-commerce-authority-center
鉴权中心

### e-commerce-common
应用公共模块，包含公共类、异常等  
+ 公共类： 
  - ServiceResult: 通用返回接口

### e-commerce-doc
文档服务-聚合各模块的文档

### e-commerce-gateway
网关服务

### e-commerce-mvc-config
web 层通用配置， 包含全局异常处理，全局返回结果封装等

### e-commerce-nacos-client
服务注册发现客户端示例模块

### e-commerce-service
业务微服务模块

#### e-commerce-account-service
账户微服务模块

#### e-commerce-goods-service
商品微服务模块

#### e-commerce-service-config
业务微服务web层通用配置

#### e-commerce-service-sdk
业务微服务模块公用sdk文件
